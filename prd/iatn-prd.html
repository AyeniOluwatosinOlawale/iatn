1. Executive Summary

IATN (International Academic Tutors Nigeria) is a comprehensive digital education ecosystem designed to become Nigeria's premier platform for international curriculum education. The platform connects students and parents with verified, high-performing tutors and schools specialising in Cambridge IGCSE, Cambridge AS & A Level, Edexcel, OxfordAQA, the International Baccalaureate Diploma, and SAT/ACT preparation.

IATN goes beyond a simple tutor directory. It functions as a verified professional marketplace, learning management platform, school discovery engine, university pathway guide, and AI-powered study companion — all in one destination.

North Star
To be the most trusted gateway to international education in Nigeria — where every student finds the right tutor, every parent makes informed school choices, and every educator builds a verifiable professional reputation.

2. Problem Statement

For Students & Parents

  • No reliable way to verify a tutor's Cambridge track record
  • No structured comparison of IGCSE/A-Level schools in Nigeria
  • No centralised source for past papers, study plans, or mock exams
  • No clear university pathway guidance post-A-Level
  • Payments are informal and unprotected

For Tutors & Schools

  • No professional identity platform for international curriculum teachers
  • No verifiable portfolio of student results
  • No structured booking, scheduling, or payment infrastructure
  • Schools cannot efficiently recruit verified Cambridge teachers
  • No CPD (continuing professional development) tracking

6. Key User Flows

Tutor Registration Flow

[Landing Page] → Click "Register as Tutor" → [Step 1: Account] Email + password + role = tutor → [Step 2: Personal Info] Name, photo, phone, state, city → [Step 3: Professional] Institution, years experience, bio, teaching mode → [Step 4: Subjects & Curricula] Select subjects + curricula + proficiency → [Step 5: Qualifications] Upload degree, teaching certs (Supabase Storage) → [Step 6: Cambridge Metrics] Students taught, pass rate, A*/A percentage → [Step 7: Rates & Schedule] Hourly rate, availability, class size → [Registration Complete] → IATN-2026-XXXXXX number generated (Supabase trigger) → Profile live (status: "Pending Verification") → Admin reviews → Verified ✓ → Badge awarded

Student Search & Booking Flow

[Search Page] → Filter: Subject + Curriculum + State + Max Price + Min Rating → [Results Grid] Tutor cards with rating, rate, verified badge → [Tutor Profile Page] Full profile, Cambridge metrics, reviews, portfolio → Click "Book a Lesson" → [Booking Calendar] Select date/time slot → [Lesson Details] Subject, curriculum, notes → [Payment] Paystack checkout (NGN) → [Confirmation] Email to both tutor and student → [Lesson Day] Video call link sent → [Post-lesson] Leave review → updates tutor rating

17. Security & Trust

Platform Security

  • All database access protected by Supabase Row Level Security (RLS) policies
  • Tutors can only read/write their own profile rows; admins bypass RLS
  • Document uploads stored in private Supabase Storage buckets (signed URLs only)
  • All payments processed server-side through Paystack; no card data touches IATN servers
  • Webhook endpoints protected with signature verification (Paystack HMAC)
  • API routes protected with Supabase session tokens
  • PII (phone, email) never exposed in public API responses

Tutor Verification Pipeline

  1. Tutor submits registration with document uploads
  2. System sends admin notification
  3. Admin reviews qualifications and identity document
  4. Admin approves → verification_status = 'verified' → badges granted
  5. Admin rejects → tutor notified with reason → can resubmit

Trust Signals Displayed to Users

  • Verified Tutor badge (green checkmark)
  • Cambridge Expert badge (blue star) for proven Cambridge track record
  • Top Tutor badge (gold) for platform rating leaders
  • Background Verified badge (shield) for tutors who opt into background check
  • School Verified badge on institution profiles
  • Exam result statistics with admin-verified label
